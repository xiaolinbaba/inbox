# 项目部署指南

在开始配置生产环境之前，请务必阅读以下说明：

> [!NOTE]
> 本项目目前仅支持 Linux 和 macOS 系统，Windows 用户建议使用 WSL2 进行操作。

> [!TIP]
> 建议在安装依赖前先更新你的包管理器，这样可以避免一些已知的版本冲突问题。

> [!IMPORTANT]
> 执行初始化脚本后，请务必记录下控制台输出的管理员临时密码，该密码仅显示一次。

> [!WARNING]
> 请勿在公共网络环境下开启调试模式（Debug Mode），这可能会导致敏感配置信息泄露。

> [!CAUTION]
> 运行 `npm run reset` 指令会永久删除数据库中的所有数据且无法撤销，请在执行前确认已做好备份。
